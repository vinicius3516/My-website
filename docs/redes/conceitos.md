# Conceitos de Redes

## Endereçamento IP

- IPv4: Ex: `192.168.0.1`
- IPv6: Ex: `2001:0db8:85a3:0000:0000:8a2e:0370:7334`

## Principais protocolos

| Protocolo | Descrição          |
|----------|--------------------|
| TCP      | Confiável, orientado à conexão |
| UDP      | Não confiável, porém rápido |
| HTTP     | Comunicação web    |
| SSH      | Acesso remoto seguro |

## Ferramentas úteis

- `ping`
- `traceroute`
- `netstat`
- `nmap`
